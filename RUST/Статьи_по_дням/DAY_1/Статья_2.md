В этой статье мы уже запрограммируем свою первую программу, и да как обычно мы будим пользоваться книгой "Программирование на RUST от Стива Клабника и Кэрол Николс". И да это выжимка, мой совет прочитать книгу целиком, но если вы хотите стартавать раньше, то можите прочесть мою статью

Давайте начинать 
Мы будим программировать игру-угадайку

Спервы создадим при помощи cargo новую 
```bash
cargo new project_2
cd project_2
```
Теперь скомпилируем и выполним нашу программу 
```bash
cargo run # быстрый запуск
cargo build # компиляция проекта в режиме --debug
```
