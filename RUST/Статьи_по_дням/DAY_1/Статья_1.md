### **Часть 1: Установка и первый запуск**

#### **Установка на Linux/macOS**
Откройте терминал и выполните:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
После установки перезагрузите оболочку или выполните:
```bash
source $HOME/.cargo/env
```
Проверьте, что всё установилось:
```bash
rustc --version  # компилятор
cargo --version  # система сборки и менеджер пакетов
```

### **Часть 2: Hello, World! — сразу в бой**

Лучший способ изучить язык — начать писать код. Rust — язык семейства C, поэтому программа начинается с точки входа `main`.

Создайте файл `hello.rs` и напишите:
```rust
// Это комментарий. Точка входа в программу — функция main.
fn main() {
    // println! — это макрос (об этом позже), который выводит строку.
    println!("Hello, World!");
}
```

**Скомпилируйте и запустите:**
```bash
rustc hello.rs
./hello
```
Вы увидите: `Hello, World!`

**Поздравляю, вы написали первую программу на Rust!** Компилятор `rustc` отлично подходит для простых скриптов.

### **Часть 3: Cargo — ваш главный инструмент**

Когда проект растёт, `rustc` становится неудобен. На помощь приходит **Cargo** — встроенная система сборки и менеджер пакетов (как `npm` для JavaScript или `go mod` для Go).

#### **Создание проекта**
```bash
cargo new my_first_app
cd my_first_app
```
Cargo автоматически создаст структуру проекта:
```
my_first_app/
├── Cargo.toml  # файл конфигурации и зависимостей
└── src/
    └── main.rs  # ваш код
```

В `src/main.rs` уже есть заготовка `Hello, world!`.

#### **Основные команды Cargo**

| Команда | Что делает |
|---------|------------|
| `cargo build` | Компилирует проект в `target/debug/`. |
| `cargo run` | Скомпилирует и сразу запустит программу. |
| `cargo check` | **Суперполезная команда!** Проверяет код на ошибки, не создавая исполняемый файл. Работает очень быстро. |
| `cargo build --release` | Собирает оптимизированную версию для продакшена (в `target/release/`). |

#### **Попробуйте прямо сейчас:**
```bash
cargo run
```
Вы увидите знакомое `Hello, world!`, но собранное через Cargo.
