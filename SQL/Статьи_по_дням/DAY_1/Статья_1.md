## Работа с базами данных
Чесно говоря мне не очень нравятся работа в СУБД, по этому я начал искать альтернативные варианты, и нашел расширение SQLTOOOlS для vs code, по факту данное расширение добовляет нужный нам функционал в vs code
Так же настоятельно прошу вас установить дополнительные расширения просто вбейте в поиск @:sqltools-driver, эти расширения являются драйверами базы данных, но мы будим в основном работать с MySql

При рассмотрении данной темы мне пришлось просмотреть кучу инфы и по итогу у меня получились следующие шаги, для того что бы запустить свою собственную базу данных

1) Первым шагом я выделю установку MySQL Server
```bash
sudo apt update
sudo apt install mysql-server
```
2) Настройка MySQL после установки
```bash
# Запуск службы MySQL (Linux/macOS)
sudo systemctl start mysql
# Или для macOS с brew:
brew services start mysql
# Проверка статуса
sudo systemctl status mysql
# Начальная настройка безопасности (Linux)
sudo mysql_secure_installation
```
3) Устанавливаем нужные расширения в vs code, т.е. SQLTools, SQLTools MySQL/MariaDB Driver
4) Настройка чоединения с MySQL и подключение через терминал для проверки
```bash
mysql -u root -p
```
5) Создание собственной базы данных
Это ключевая вещь
```sql
-- Создадим новую базу данных
CREATE DATABASE my_project_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
-- Создаем пользователя для работы с этой БД
CREATE USER 'project_user'@'localhost' IDENTIFIED BY 'StrongPassword123!';
-- Даем все права на созданную БД новому пользователю
GRANT ALL PRIVILEGES ON my_project_db.* TO 'project_user'@'localhost';
-- Применяем изменения привилегий
FLUSH PRIVILEGES;
-- Выходим из MySQL
EXIT;
```
6) Настройка SQLTools, откройте панель SQLTools и добавте новое соединение выбрав при этом же MySQL/MariaDB
7) Заполняем информацию
```json
{
  "name": "My Project Database",           // Произвольное имя соединения
  "driver": "MySQL",                       // Тип драйвера
  "server": "localhost",                   // Сервер (localhost для локальной БД)
  "port": 3306,                           // Стандартный порт MySQL
  "database": "my_project_db",            // Имя созданной базы данных
  "username": "project_user",             // Имя пользователя
  "password": "StrongPassword123!",       // Пароль пользователя
  "connectionTimeout": 30000,             // Таймаут подключения в миллисекундах
  "askForPassword": false                 // Не спрашивать пароль каждый раз
}
```
Мои поздравления мы настроили базу данных
